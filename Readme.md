# Ros2Go2Estimator 🦾
[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

基于纯运动学的双足/四足机器人位置估计算法，提供高精度里程计解决方案。

## 📚 项目亮点
- 四足站立行走误差率1%  
- 动态行走模式误差率0.5%
- 支持380米长距离定位
- EtherCAT/CANopen双协议兼容
